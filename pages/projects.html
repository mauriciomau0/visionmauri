Gait Stability Prediction Using Computer Vision and Synthetic Data

Year:
2025
Focus:
Gait stability assessment in older adults using computer vision and synthetic data generation
Population:
Older adults (n=14)

Led research developing a novel approach for gait stability assessment in older adults using computer vision and synthetic data generation. This work addressed critical challenges in fall risk monitoring by creating accessible tools for gait stability prediction.

Research Highlights:

  • Implemented MediaPipe pose tracking to extract body landmarks from frontal plane older adults' walking videos
  • Developed a constraint-based synthetic data generation methodology that preserved variable relationships
  • Achieved exceptional synthetic data fidelity (97.09% overall) while maintaining biomechanical variable correlations
  • Demonstrated that models trained exclusively on synthetic data outperformed those trained on real data alone for margin of stability prediction

This project will result in a publication you will find in the publication section in the future.

Synthetic Data-Driven Machine Learning for Performance Prediction

Year:
2024
Technology:
Tabular Variational Autoencoders (TVAE)
Dataset:
Athlete performance dataset (n=41)

Researched the application of synthetic data generation (SDG) to address data scarcity challenges in athlete performance monitoring. This work investigated how Tabular Variational Autoencoders (TVAE) can generate high-quality synthetic data that preserves the statistical properties and relationships in a real athlete performance dataset.

Research Highlights:

  • Developed a comprehensive evaluation framework combining column shape similarity metrics (85.53%) and Hellinger distance analysis (0.169)
  • Implemented a two-phase approach to assess synthetic data utility: augmenting real data with synthetic samples and training models exclusively on synthetic data
  • Demonstrated that models trained with synthetic data could outperform real-data baselines across multiple performance metrics
  • Identified optimal synthetic data proportions for each performance metric classification task

This project will result in a publication you will find in the publication section in the future.

Machine Learning-Based Performance Monitoring for Athletes

Year:
2023
Sport:
Gaelic Football
Dataset:
Athletes (n=41)

Developed a comprehensive machine learning approach for predicting performance attenuation in Gaelic football athletes. This research addressed the challenge of monitoring complex interactions between physical capabilities and performance decline during and after a competitive match.

Research Highlights:

  • Implemented a novel ranking system to quantify performance attenuation based on pre-post match differences
  • Applied dimensionality reduction techniques (PCA) to identify key components influencing performance
  • Evaluated multiple machine learning algorithms (Random Forest, AdaBoost, XGBoost) with various data balancing techniques
  • Achieved high predictive accuracy for multiple performance rankings, with F1-scores ranging up to 0.90

This project will result in a publication you will find in the publication section in the future.

Machine Learning/Augmented Reality Immersive Training System

Year:
2022
Technology:
AR, ML, Wearable Sensors
Status:
Published in MMVE '22

Contributed to developing an augmented reality-based training system designed to improve athletic performance monitoring and decision-making processes. This innovative proof-of-concept system combines wearable physiological sensors with immersive AR feedback to track performance attenuation in athletes during training.

Key Contributions:

  • Design a multimodal approach to immersive training that captures both behavioural and biomechanical factors
  • Integrate physiological monitoring (heart rate, electrodermal activity) with motion analysis using IMU sensors
  • Create a real-time visual feedback system that provides athletes with real-time performance insights
  • Develop a scoring mechanism that helps athletes understand their physical limits and fatigue patterns

Fall Detection System for Public Health

Year:
2018
Technology:
IMU sensors, k-Nearest Neighbor algorithms
Accuracy:
99.4%

My team and I developed a machine-learning framework for fall detection in elderly populations using low-cost inertial sensors. This research addressed a significant public health challenge in Brazil, where falls represent a major cause of injury, hospitalisation, and healthcare expenditure among older adults.

Key Activities:

  • Designed and implemented a wearable system using IMU (Inertial Measurement Unit) sensors to detect and classify different fall states
  • Applied machine learning techniques to identify fall patterns with 99.4% accuracy using k-Nearest Neighbor algorithms
  • Conducted validation trials with real participants to test and refine the system
  • Developed a solution that can detect falls in real-time to enable faster intervention

This project resulted in a publication that you can find in the publication section.
