Gait Stability Prediction Using Computer Vision and Synthetic Data | 2025

I led research developing a novel approach for gait stability assessment in older adults using computer vision and synthetic data generation. This work addressed critical challenges in fall risk monitoring by creating accessible tools for gait stability prediction.

Research Highlights:

  1. Implemented MediaPipe pose tracking to extract body landmarks from frontal plane older adults' walking videos (n=14)
  2. Developed a constraint-based synthetic data generation methodology that preserved variable relationships
  3. Achieved exceptional synthetic data fidelity (97.09% overall) while maintaining biomechanical variable correlations
  4. Demonstrated that models trained exclusively on synthetic data outperformed those trained on real data alone for margin of stability prediction

Synthetic Data-Driven Machine Learning for Performance Prediction | 2024

I researched the application of synthetic data generation (SDG) to address data scarcity challenges in athlete performance monitoring. This work investigated how Tabular Variational Autoencoders (TVAE) can generate high-quality synthetic data that preserves the statistical properties and relationships in a real athlete performance dataset (n=41).

Research Highlights:

  1. Developed a comprehensive evaluation framework combining column shape similarity metrics (85.53%) and Hellinger distance analysis (0.169)
  2. Implemented a two-phase approach to assess synthetic data utility: augmenting real data with synthetic samples and training models exclusively on synthetic data
  3. Demonstrated that models trained with synthetic data could outperform real-data baselines across multiple performance metrics
  4. Identified optimal synthetic data proportions for each performance metric classification task

Machine Learning-Based Performance Monitoring for Athletes | 2023

I developed a comprehensive machine learning approach for predicting performance attenuation in Gaelic football athletes (n=41). This research addressed the challenge of monitoring complex interactions between physical capabilities and performance decline during and after a competitive match.

Research Highlights:

  1. Implemented a novel ranking system to quantify performance attenuation based on pre-post match differences
  2. Applied dimensionality reduction techniques (PCA) to identify key components influencing performance
  3. Evaluated multiple machine learning algorithms (Random Forest, AdaBoost, XGBoost) with various data balancing techniques
  4. Achieved high predictive accuracy for multiple performance rankings, with F1-scores ranging up to 0.90

Machine Learning/Augmented Reality Immersive Training System | 2022

I contributed to developing an augmented reality-based training system designed to improve athletic performance monitoring and decision-making processes. This innovative proof-of-concept system combines wearable physiological sensors with immersive AR feedback to track performance attenuation in athletes during training. Some of the highlights involved:

  1. Design a multimodal approach to immersive training that captures both behavioural and biomechanical factors
  2. Integrate physiological monitoring (heart rate, electrodermal activity) with motion analysis using IMU sensors
  3. Create a real-time visual feedback system that provides athletes with real-time performance insights
  4. Develop a scoring mechanism that helps athletes understand their physical limits and fatigue patterns

Fall Detection System for Public Health | 2018

My team and I developed a machine-learning framework for fall detection in elderly populations using low-cost inertial sensors. This research addressed a significant public health challenge in Brazil, where falls represent a major cause of injury, hospitalisation, and healthcare expenditure among older adults. Some of activities involved:

  1. Designed and implemented a wearable system using IMU (Inertial Measurement Unit) sensors to detect and classify different fall states
  2. Applied machine learning techniques to identify fall patterns with 99.4% accuracy using k-Nearest Neighbor algorithms
  3. Conducted validation trials with real participants to test and refine the system
  4. Developed a solution that can detect falls in real-time to enable faster intervention
